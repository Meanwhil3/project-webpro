การรันเว็บไซต์
ระบบปฏิบัติการ Windows ให้ทำการคลิ้กขวาที่ไฟล์ run.bat จากนั้นกด Run as administrator
ระบบปฏิบัติการ MacOS ให้ทำการคลิ้กขวาที่ folder ของโปรเจ็ค กดเปิด New terminal at folder
พิมพ์คำสั่ง chmod +x run.command จากนั้นดับเบิ้ลคลิ้กที่ไฟล์ run.command ได้เลย
